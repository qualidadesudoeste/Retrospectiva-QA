# Ajustes para Minimalismo Ultra

## Problemas Identificados

1. **Faixa azul grande** no header "Análise de Qualidade" - muito colorida
2. **Cards secundários** ainda sem interatividade
3. **Cores muito saturadas** - precisa ser mais neutro
4. **Gradientes** ainda presentes em alguns elementos

## Ajustes Necessários

### 1. Headers/Títulos
- Remover fundo azul/roxo dos headers
- Deixar apenas texto com cor neon sutil
- Fundo totalmente transparente ou muito escuro

### 2. Redução de Cores
- Manter fundo escuro neutro em TUDO
- Cores neon APENAS em:
  - Bordas finas dos cards principais
  - Ícones pequenos
  - Texto de destaque pontual
- Remover todos os gradientes coloridos

### 3. Interatividade
- Adicionar hover em TODOS os cards
- Adicionar cursor pointer
- Efeitos sutis de elevação

### 4. Elegância
- Espaçamentos maiores
- Bordas mais finas
- Sombras mais sutis
- Tipografia refinada
